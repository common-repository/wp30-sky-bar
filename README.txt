"WP30 Sky Bar" is a top bar for your message channel at any WordPress site.

== Description ==

Inspired by This is 2nd plugin from Challenge "WP30" by Terry Tsang. It aims to give you the message channel for marketing.


There are 4 option tabs:

*   "General" General settings for Sky Bar
*   "Content" options to control what to show in sky bar
*   "Design" options to control how to show sky bar
*   "Rules" options to control when to show sky bar

 
== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `wp30-sky-bar` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. There are new side menu "Sky Bar" and select the submenu "All Sky Bars" and "Add New"
